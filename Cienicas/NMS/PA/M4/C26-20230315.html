Propiedades de la funcin logartmica
1. S a>1 la funcin es positiva para toda x >1 y negativa para toda x<1. la funcin no est definida para
valores negativos de x.
2. S a>1 la funcin es simplemente creciente. Si x crece la y crece.
3. S a<1, la funcin es negativa para toda x>1 y positiva para toda x<1. La funcin no est definida para
valores negativos de x.
Logaritmo base 1/2 de x
4. S a<1, la funcin es siempre decreciente. Si x crece y decrece.
5. Si a>1 a<1, la grfica interseca al eje X en (1,0).
"Logaritmo base a de x igual con y"
