Propiedades de la función logarítmica
1. Sí a>1 la función es positiva para toda x >1 y negativa para toda x<1. la función no está definida para
valores negativos de x.
2. Sí a>1 la función es simplemente creciente. Si x crece la y crece.
3. Sí a<1, la función es negativa para toda x>1 y positiva para toda x<1. La función no está definida para
valores negativos de x.
